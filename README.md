Usage:

Just add the following to your pom.xml to use it in your projects:

        <repository>
           <id>jitpack.io</id>
           <url>https://jitpack.io</url>
        </repository>
        
        
        <dependency>
            <artifactId>LanguageAPI</artifactId>
            <version>0.3</version>
        </dependency>
