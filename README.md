Usage:

Just use the following pom.xml to use it in your projects:

        <repository>
           <id>jitpack.io</id>
           <url>https://jitpack.io</url>
        </repository>
        
        
        <dependency>
            <artifactId>LanguageAPI</artifactId>
            <version>0.2</version>
            <version>0.03</version>
        </dependency>
